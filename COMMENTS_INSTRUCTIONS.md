# Включение комментариев в канале

**Важно:** В Telegram Bot API нет метода, который позволяет боту включить обсуждение (комментарии) у канала. Это делается только владельцем канала в приложении.

## Что сделать вручную (1–2 минуты)

1. **Создайте группу** в Telegram (если ещё нет): «Обсуждение канала @online_school_online» или любое название.
2. **Откройте канал** @online_school_online → Настройки (шестерёнка) → **Обсуждение** (Discussion).
3. **Привяжите группу:** выберите созданную группу — комментарии к постам канала появятся в этой группе (под каждым постом будет ссылка «Обсуждение»).
4. **Добавьте бота в группу обсуждения** администратором с правом «Публикация сообщений», чтобы он мог отвечать на комментарии от имени канала.

После этого все **новые** посты в канале будут с комментариями. Старые посты, опубликованные до привязки группы, комментариев не получат.

Когда обсуждение включено — в `.env` можно поставить `COMMENTS_ENABLED=true`, тогда в новых постах можно снова использовать призывы к обсуждению в комментариях.
